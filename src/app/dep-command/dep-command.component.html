Deprecated Features

  • purge command (deprecated in 6.9.1) - removed due to Discord API limitations on the age at which messages could no longer be bulk deleted.
  • inulove (deprecated in 6.10.2) - removed as incomplete, not used enough to warrant completion.
  • inuquote (deprecated in 6.10.2) - removed as incomplete, not used enough to warrant completion. quote.
  • optout (deprecated in 7.0.8) - Removed as unused, unneeded for GDPR compliance.
  • block (deprecated in 7.1.0) - Removed as blocking users became server-specific.
